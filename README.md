# AI Resume Analyzer

## Overview
AI Resume Analyzer is a Python project that analyzes a user's resume based on their skills, education, and experience.

## Features
- Resume skill analysis
- Resume scoring
- Career recommendations
- Job recommendations
- Improvement suggestions
- Saves a resume analysis report

## Technologies Used
- Python
